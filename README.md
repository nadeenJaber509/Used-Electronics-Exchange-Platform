# Used Electronics Exchange Platform

A complete Software Engineering course project developed for **COMP433 – Software Engineering** at Birzeit University.

The project presents the full software engineering lifecycle of a web-based platform for buying, selling, exchanging, repairing, and managing used electronic devices. It covers business analysis, requirements engineering, UML modeling, system design, and software architecture.

---

## 📌 Project Overview

The platform is designed to digitalize the workflow of a used electronics business by replacing manual processes with an integrated web-based system.

Main services include:

- Device purchasing
- Device sale & exchange
- Repair request management
- Appointment scheduling
- Refund processing
- Device inspection
- Delivery management
- Report generation
- User and employee management
